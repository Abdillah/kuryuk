# Kuryuk
Rooster-bank that sound. Money manager API built using Rust.

![License: MIT](https://img.shields.io/badge/license-MIT-green)

## Planned Features
* Basic transaction records
* Wallets
* Multi-users
* Plugins for external API sync (bank, virtual money, etc)
